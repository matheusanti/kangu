Integração Kangu e WooCommerce

== Description ==

A Kangu é uma plataforma gratuita que conecta sua loja com os Correios e diversas transportadoras sem necessidade de contrato, mensalidade ou envio mínimo.

Quem tem integração com a Kangu, **economiza até 75% no valor do frete e pode postar em milhares de pontos de envio** espalhados pelo Brasil e agências dos Correios.

Você pode testar de graça: usando o cupom exclusivo para clientes WooCommerce, WOO30KANGU, você tem R$ 30 para fazer qualquer envio. Válido para novos usuários

= Com a Kangu sua loja tem: =
- Opções de entrega e retira no checkout para o seu cliente
- Diversas transportadoras à sua disposição

= Confira outros benefícios =
- **Cotação Simultânea** e sem custo
- **Ampla rede de transportadoras:** Garantimos a entrega para todo o Brasil. Com diversas transportadoras, como Correios, Loggi, Carbono Zero, Rede Sul e Sequoia
- **Sem volume mínimo** de pedidos e sem contrato. Você só paga quando usar o serviço
- **Controle e gestão:** Em um painel exclusivo, você tem controle e rastreamento dos seus envios do início ao fim
- **Comodidade para postar:** Mais de 2.600 Pontos Kangu, além das agências dos Correios. Os pontos são comércios de bairro capacitados para receber, armazenar e enviar, com todo cuidado e segurança. Basta uma caminhada pelo seu bairro para enviar seus pedidos

= Benefícios do plugin =
- Instalação e configuração simplificada. Com poucos cliques você já pode começar a economizar no frete!
- Cálculo do frete na página do produto com opções de envio e retira Kangu
- Listagem dos fretes de uma forma mais organizada
- Integração automática com a plataforma Kangu
- Cotador de fretes pós-venda
- Rastreio automático dos pedidos com exibição das ocorrências no admin e na conta do usuário
- Possibilidade de adicionar prazo e valor sobre os fretes

Kangu é uma nova forma de fazer entregas. Conectamos e-commerces, clientes, transportadoras e pontos de envios, para oferecer envios de **menor custo, convenientes e sustentáveis**.
Para pequenos ou médios volumes, para bem pertinho ou longas distâncias, vai de Kangu!

**Instale agora!**

== Installation ==

= Requerimentos: =

- Plugin [Brazilian Market on WooCommerce](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil)
- Estar cadastrado na plataforma [Kangu](https://www.kangu.com.br/woocommerce)
- WooCommerce 3.0 ou posterior

= Instalação do plugin: =

- Remova a versão antiga do plugin Kangu Frete e Kangu Frete sobre o Produto (caso você tenha instalado)
- Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.
- Ative o plugin.
- Acesse o menu WooCommerce > Fretes Kangu
- Clique no botão Autenticar APP
- Faça login na sua conta Kangu ou cadastre-se
- Configure o plugin

= Configurações do plugin: =

- Por padrão, todas as configurações estarão habilitaddas, porém você pode desabilita-las a qualquer momento:
- Status do plugin
- Cálculo do frete na página do produto
- Ocultar o campo País no carrinho
- Ocultar o campo Cidade no carrinho
- Ocultar o campo Estado no carrinho

= Configurações na plataforma Kangu =
- Habilitar ou desabilitar transportadoras que serão disponibilizadas
- Habilitar ou desabilitar o Retira Kangu
- Configurar prazo adicional sobre o frete
- Configurar preço adicional sobre o frete

== Frequently Asked Questions ==

= Qual é a licença do plugin? =

Este plugin esta licenciado como GNU General Public License v3.0.

= O que eu preciso para utilizar este plugin? =

* Plugin [Brazilian Market on WooCommerce](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil)
* Estar cadastrado na plataforma [Kangu](https://www.kangu.com.br/woocommerce)
* WooCommerce 3.0 ou posterior

= Onde configuro os métodos de entrega? =

Você pode configurar as transportadoras em [Minhas Preferências](https://www.portal.kangu.com.br/minhas-preferencias).

= Tenho que pagar alguma mensalidade para usar a Kangu? =

A Kangu não cobra comissões ou mensalidades, também não precisa de nenhum contrato. Você só paga pelos envios que realizar por meio da Kangu

= A Kangu é uma transportadora? =

Não, a Kangu é uma rede que conecta transportadoras, lojistas e pontos de envio
